concluiRxWMSCorteOperPedido
^^^^^^^^^^^^^^^^^^^^^^^^^^^

- Descripción: Concluye el románico, actualizando la fecha de exportación;
- Entrada: ID de pedido que se actualizará;
- Parámetros (1):
   - Código de pedido en el sistema WMS.
   - Tipo: entero.
   - Campo obligatorio.
